# Specification

## Summary
**Goal:** Add update functionality to the flight log form for editing existing entries.

**Planned changes:**
- Add an "Update" button to the flight log form that appears when editing an existing entry
- Position the Update button alongside the existing submit button
- Implement update mutation call with loading state and success notification
- Ensure date input field remains editable in edit mode with calendar picker functionality
- Verify close button properly cancels edit operation without saving changes

**User-visible outcome:** Users can edit existing flight log entries using an Update button that saves changes to the backend, with proper feedback and the ability to cancel edits without saving.
